## 🌟 快速参考卡

### 核心概念
- **Flash 筛选**: 快速评分（0-100），默认阈值 50
- **Pro 分析**: 深度分析（0-100），默认阈值 70
- **流式处理**: 边搜索边筛选边导出
- **岗位扩展**: AI 自动生成岗位变体
- **严格地点匹配**: 只看当前所在地

### 核心 API
```python
# 一站式处理
quick_streaming_pipeline(user_input, ...)

# 搜索
SerperSearch().search_linkedin_profiles(...)

# 筛选
DetailedScreening().screen_candidates_two_stage(...)

# 导出
GoogleSheetsExporterOAuth().export_candidates(...)
```

### 配置文件
- `config.py` - 系统配置
- `llm_config.py` - LLM 配置
- `.env` - 环境变量
- `oauth_credentials.json` - OAuth 凭证
